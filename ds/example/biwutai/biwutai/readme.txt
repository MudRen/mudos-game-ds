這是一個自動的比武系統，玩家報名參加比武后，程序將自動分組
并判斷勝負。

具體安裝方法是：
1。解壓縮后會有一個/biwu目錄，把這個目錄移動到/d/目錄下。
2。把/biwu目錄下的message.c文件移動到/adm/simul_efun/
	目錄下。
3。在damage.c文件中的void unconcious()函數和void die()函數
	中加上

	if (environment()->query("biwutai")) {
		environment()->no_die(this_object());
		return;
	}

4。然后重新啟動，這就可以了。

具體操作方法是：
比武方式分為四種
□比武方式□●個人賽    ●擂台賽        ●門派賽        ●循環賽
巫師可以用指令切換

個人賽 biwu geren	開放比武場隨意使用，玩家可以任意上台比武。
擂台賽 biwu leitai	（該功能留待擴展）
門派賽 biwu menpai	單個門派內部比武，獲勝者將獲得掌門大弟子稱號！
循環賽 biwu xunhuan	所有玩家都能參加。

個人賽很簡單，所有人都可以上台比武，比武一律使用KILL指令，但失敗的玩家
	不會死。其他玩家可以在比武台下觀看比武。
門派賽玩家可以在公平子處用menpai指令提出本門比武的申請。巫師可以用
	agree同意、del刪除、dissent不同意。
	指令格式：比如要同意天羽派5月20日比武就用
	(agree 天羽派 5越20日 星期六比武)
	然后可以用list 指令查詢結果

	當巫師批准后申請人自動就報了名，其他玩家可以到公平子處用canjia
	指令報名。
巫師宣布比武開始指令是start。如果要宣布門派比武開始用(start <門派中文名>)
巫師宣布比武開始后，公平子自動對在線的已報名參加比武的玩家數據進行處理
每四人分成一組進行循環賽。這時巫師就可以到比武台貴賓席查看比武進程了。
巫師在貴賓席可以用check指令查看比武進程。
用urged指令將遲遲不上台而影響比武進程的玩家拉上台。
用xsave指令儲存比武進程。
用next指令進行下一回合比武。(next指令不要隨意使用，因為這將影響比武的公平)
用next <玩家id>讓這一回合該玩家獲勝。繼續下一回合。
